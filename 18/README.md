## Vagrant

### Задание

#### 1. Подготовка окружения:

<ul>
  <li>Создать директорию для проекта.</li>
  <li>Создать базовую виртуальную машину:</li>
  <li>Настроить память ВМ: 1024 МБ.</li>
  <li>Добавить пару виртуальных диска размером 1 ГБ каждый.</li>
  <li>Настроить проброс 80 порта с гостевой системы на порт 8080 хостовой системы.</li>
  <li>Провижининг:</li>
   <ul>
     <lli>Форматирует добавленные диски в файловую систему ext4.</lli>
     <lli>Создает точки монтирования /mnt/disk1 и /mnt/disk2.</lli>
     <lli>Монтирует диски в указанные директории.</lli>
     <lli>Добавляет записи в /etc/fstab для автоматического монтирования при загрузке.
</lli>
   </ul>
</ul>
